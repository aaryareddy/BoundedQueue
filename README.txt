Inputs and procedure to run the program:

The size of the list can be passed as an argument.However, if no argument is passed, a list of size 10 is created.
Once you compile and execute(run) the code, the list is printed.
There are 7 options - 1, 2, 3, 4, 5, 6 AND 7

1 - Offer elemnt to the rear position of the queue 
2 - Poll front element from the queue
3 - Display front element of the queue
4 - display size of the queue
5 - checks if the queue is empty or not
6 - Clear the queue
7 - Copies the queue elements to array from front to rear order
